# pure

[![Slack Room][slack-badge]][slack-link]

Yet another port of the ZSH pure theme to fish. It's not intended to be customizable, basically all values are hard-coded to my personal preference.

![pure]

## Install

With [fisherman]

```
fisher MaxMilton/pure
```
